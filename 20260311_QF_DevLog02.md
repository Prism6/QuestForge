### C. 버그 수정

#### C-1. `_parse_json_response()` JSON 파싱 강화 (`utils/quest_generator.py`)

**문제:**
- Claude 응답에 JSON 앞뒤로 부가 텍스트가 있을 경우 (`Here's your quest:\n```json\n...\n```) 파싱 실패
- 기존 코드는 `startswith("```json")` 방식이라 앞에 텍스트가 있으면 처리 불가

**해결:**
3단계 fallback 전략 도입:
1. regex로 코드 블록 내 JSON 추출 (```` ```(?:json)?...\``` ````)
2. 코드 블록 마커만 제거
3. `{` / `}` 위치 기반으로 JSON 객체 범위 직접 추출

```python
# 1차: 코드 블록 패턴
json_block_match = re.search(r'```(?:json)?\s*(\{[\s\S]*?\})\s*```', text)
# 3차: 브레이스 범위 추출
start = text.find('{')
end = text.rfind('}')
```

#### C-2. `create_regeneration_prompt()` JSON 스키마 누락 (`utils/prompts.py`)

**문제:**
- 재생성 프롬프트가 "이전과 동일한 JSON 구조"라고만 명시 → 대화 히스토리 없이 단발 호출이므로 모델이 스키마를 알 수 없음
- `description` 등 필드가 누락된 응답이 반환될 가능성

**해결:**
- 원본 `create_quest_prompt()`와 동일한 JSON 스키마 템플릿을 재생성 프롬프트에 명시
- `difficulty`, `quest_type` 값을 템플릿에 직접 삽입하여 일관성 보장

### B. v1.1 개선

#### B-1. 프롬프트 튜닝 (`utils/prompts.py`)

**장르별 톤 가이드 추가:**
```python
genre_tone_guide = {
    "RPG": "고전적인 판타지 서사, 영웅의 여정, 선악 구도",
    "액션": "긴박한 전투 상황, 짧고 강렬한 대사, 즉각적인 보상",
    "시뮬레이션": "일상적이고 현실적인 문제, 경제/관계 중심, 차분한 톤",
    "어드벤처": "탐험과 발견, 신비로운 분위기, 호기심을 자극하는 내용",
    "MMORPG": "대규모 세계관, 길드/파티 요소, 반복 플레이를 고려한 보상",
    "로그라이크": "절차적 생성 느낌, 간결한 설명, 높은 위험/높은 보상",
}
```

**`description` 필드 추가:**
- 퀘스트 배경 스토리를 2~3문장으로 요약하는 필드
- 퀘스트 미리보기 UI 기능의 데이터 소스

**퀘스트 타입 가이드 강화:**
- 기존보다 세부적인 스토리 방향 제시 (예: 메인은 "세계관에 영향을 주는 사건")

#### B-2. UI 스타일링 + 퀘스트 미리보기 (`app.py`)

**커스텀 CSS 추가:**
- 퀘스트 description을 표시하는 다크 테마 카드 컴포넌트
- 섹션 구분선, 퀘스트 타입 뱃지 스타일

**퀘스트 미리보기 카드:**
- description 필드가 있을 경우 퀘스트 헤더 아래 배경 스토리 카드 표시

**난이도 표시 개선:**
- 숫자 별(⭐ 반복) → 아이콘 + 채워진/빈 별 조합 (`🟢 ★★☆☆☆`)
- 난이도별 색상 아이콘: 🟢(1) 🟡(2) 🟠(3) 🔴(4) 💀(5)

**퀘스트 타입 한글 표시:**
- `main` → `메인`, `sub` → `서브` 등 API 값을 한글로 변환하여 UI 표시

#### B-3. 에러 로깅 시스템 (`app.py`, `utils/quest_generator.py`)

**`logging` 모듈 적용:**
- `app.py`: 앱 초기화, 초기화 실패 이벤트 로깅
- `quest_generator.py`: API 요청/응답, JSON 파싱, 성공/실패 이벤트 로깅

```python
logging.basicConfig(
    level=logging.INFO,
    format="%(asctime)s [%(levelname)s] %(name)s - %(message)s",
)
```

```
2026-03-11 10:23:04 [INFO] utils.quest_generator - 퀘스트 생성 성공: 어둠의 사제단을 막아라
```

#### B-4. `data_exporter.py` 개선

**`description` 컬럼 추가:**
- Excel 출력에 description 컬럼 포함 (quest_name 다음 위치)

**컬럼 인덱스 버그 수정:**
- `chr(65 + idx)` 방식은 26개 컬럼(Z) 초과 시 잘못된 문자 반환

## 이슈 및 해결

### 이슈 1: 재생성 프롬프트 설계 결함
**문제:** 단발 API 호출 구조에서 "이전과 동일한 형식"은 의미 없음
**해결:** JSON 스키마 템플릿 명시 + 동적 값(difficulty, quest_type) 직접 삽입

### 이슈 2: Excel 컬럼 인덱스 한계
**문제:** `chr(65 + idx)`는 26개 컬럼 초과 시 잘못된 문자 생성 (예: `[`, `\` 등)
**해결:** `openpyxl.utils.get_column_letter()` 사용으로 임의의 컬럼 수 지원
