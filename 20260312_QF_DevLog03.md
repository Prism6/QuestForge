## 4. 2차 개선 (v1.3) — DIP · Strategy · 도메인 모델 도입

### 4-1. 신규 파일

#### `utils/models.py` — 퀘스트 도메인 모델

```python
@dataclass class NPC: name, location
@dataclass class Objective: type, target, location, count
@dataclass class Rewards: gold, exp, items
@dataclass class Dialogue: accept, progress, complete

@dataclass
class QuestData:
    # 모든 퀘스트 필드
    @classmethod
    def from_dict(cls, data: Dict) -> "QuestData": ...  # 필수 필드 누락 시 ValueError
    def to_dict(self) -> Dict: ...
```

- `Dict.get("key", default)` 패턴 → 속성 접근(`quest.npc.name`)으로 전환
- `QuestData.from_dict()` 가 `validate_quest_data()` 역할 대체 (fail-fast)

#### `utils/llm_client.py` — LLMClient 추상 인터페이스 (DIP)

```python
class LLMClient(ABC):
    @abstractmethod
    def complete(self, prompt: str) -> str: ...

class AnthropicLLMClient(LLMClient):
    MODEL = "claude-sonnet-4-20250514"
    MAX_TOKENS = 2000
    TEMPERATURE = 1.0
    def complete(self, prompt: str) -> str: ...  # Anthropic API 호출
```

- `QuestGenerator`가 `LLMClient` 인터페이스에만 의존 → 공급자 교체 시 `QuestGenerator` 무수정

#### `utils/export_strategy.py` — Strategy 패턴 (OCP + 다형성)

```python
class ExportStrategy(ABC):
    @property @abstractmethod def mime_type(self) -> str: ...
    @property @abstractmethod def file_extension(self) -> str: ...
    @abstractmethod def export(self, quests: List[QuestData]) -> bytes: ...
    def generate_filename(self, prefix: str = "quest") -> str: ...  # 공통 구현

class JsonExportStrategy(ExportStrategy): ...
class ExcelExportStrategy(ExportStrategy):
    COLUMN_ORDER, MAX_COLUMN_WIDTH, SHEET_NAME = [...]
    @staticmethod def _flatten(quest: QuestData) -> dict: ...
```
