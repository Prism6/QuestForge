### 2. 유틸리티 모듈 구현

#### 2.1 utils/prompts.py
**목적:** Claude API에 전달할 프롬프트 템플릿 관리

**주요 기능:**
- `create_quest_prompt()`: 장르/테마/난이도/타입 기반 퀘스트 생성 프롬프트
- `create_regeneration_prompt()`: 기존 퀘스트 재생성 프롬프트

**특징:**
- 난이도별 보상 차등 적용 로직
- 퀘스트 타입별 가이드라인 제공
- JSON 구조화 출력 강제

**코드 구조:**
```python
def create_quest_prompt(genre, theme, difficulty, quest_type) -> str
def create_regeneration_prompt(original_quest, feedback="") -> str
```

#### 2.2 utils/quest_generator.py
**목적:** Claude API 호출 및 퀘스트 생성 로직

**주요 기능:**
- `QuestGenerator` 클래스
  - `generate_quest()`: 새 퀘스트 생성
  - `regenerate_quest()`: 퀘스트 재생성
  - `validate_quest_data()`: 퀘스트 데이터 검증
  - `_parse_json_response()`: JSON 파싱 (마크다운 제거)

**특징:**
- 환경변수 및 Streamlit Secrets에서 API 키 로드
- Claude Sonnet 4 모델 사용 (claude-sonnet-4-20250514)
- JSON 파싱 에러 처리
- 필수 필드 검증 로직

**에러 처리:**
- API 호출 실패
- JSON 파싱 실패
- API 키 미설정

#### 2.3 utils/data_exporter.py
**목적:** 퀘스트 데이터 JSON/Excel 내보내기

**주요 기능:**
- `DataExporter` 클래스
  - `to_json()`: JSON 문자열 변환
  - `to_excel()`: Excel 파일(BytesIO) 변환
  - `generate_filename()`: 타임스탬프 기반 파일명 생성
  - `create_download_data()`: 다운로드용 데이터 준비

**특징:**
- 중첩 JSON을 평면 데이터로 변환 (Excel용)
- 컬럼 순서 정의 및 너비 자동 조정
- 한글 데이터 UTF-8 인코딩 처리

**엑셀 컬럼 구조:**
```
quest_id, quest_name, quest_type, difficulty, genre, theme,
npc_name, npc_location, objective_type, objective_target,
reward_gold, reward_exp, reward_items, dialogue_accept, ...
```

### 3. 메인 애플리케이션 (app.py)

#### 3.1 주요 기능
- **세션 상태 관리**
  - 퀘스트 히스토리 저장
  - 현재 퀘스트 추적
  - QuestGenerator 인스턴스 캐싱

- **사이드바 UI**
  - 장르 선택 (6가지: RPG, 액션, 시뮬레이션, 어드벤처, MMORPG, 로그라이크)
  - 테마 입력 (자유 텍스트)
  - 난이도 슬라이더 (1-5)
  - 퀘스트 타입 (메인/서브/일일/반복)
  - 생성 히스토리 표시 (최근 5개)

- **메인 영역**
  - 퀘스트 정보 표시
    - 헤더 (퀘스트명)
    - 기본 정보 (장르, 타입, 난이도, 보상)
    - NPC 정보
    - 목표 (objective)
    - 보상 (골드, 경험치, 아이템)
    - 대사 (수락/진행/완료)

  - 액션 버튼
    - JSON 다운로드
    - Excel 다운로드 (전체 히스토리)
    - 재생성
    - 히스토리 초기화

#### 3.2 UI/UX 특징
- Streamlit 기본 UI 활용
- 한국어 인터페이스
- 메트릭을 통한 시각적 정보 표시
- Expander를 통한 대사 접기/펼치기
- 에러 메시지 사용자 친화적 표시

#### 3.3 에러 처리
- API 키 미설정 시 가이드 메시지
- 테마 미입력 경고
- 퀘스트 생성 실패 시 에러 메시지
- 검증 실패 시 알림

## 기술적 결정 사항

### 1. Claude API 모델 선택
- **선택:** Claude Sonnet 4 (`claude-sonnet-4-20250514`)
- **이유:**
  - 한국어 품질 우수
  - 구조화 출력 안정적
  - JSON 파싱 정확도 높음
  - 비용 대비 성능 우수

### 2. JSON 파싱 전략
- 마크다운 코드 블록 제거 로직 구현
- `json.loads()` 사용 전 텍스트 전처리
- 에러 발생 시 원본 응답 포함하여 디버깅 용이

### 3. 데이터 검증
- 필수 필드 존재 여부 확인
- 중첩 객체 타입 검증
- 검증 실패 시 사용자에게 알림

### 4. Streamlit 세션 상태 활용
- `st.session_state`를 통한 히스토리 관리
- 페이지 새로고침 없이 상태 유지
- Generator 인스턴스 재사용으로 성능 최적화
