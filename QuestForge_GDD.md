# QuestForge - AI 퀘스트 생성기

## 1. 프로젝트 개요

### 1.1 프로젝트 정보
| 항목 | 내용 |
|------|------|
| 프로젝트명 | QuestForge |
| 버전 | 1.0.0 |
| 작성자 | 한상범 |
| 작성일 | 2025.03 |
| 기술 스택 | Python, Streamlit, Claude API |

### 1.2 목적
게임 기획자의 퀘스트/이벤트 기획 업무를 AI로 보조하여 **기획 생산성을 향상**시키는 웹 기반 툴

### 1.3 타겟 유저
| 유저 유형 | 사용 목적 |
|----------|----------|
| 게임 기획자 | 퀘스트 초안 빠른 생성, 아이디어 발상 |
| 인디 개발자 | 1인 개발 시 기획 리소스 절감 |
| 게임 PD/디렉터 | 퀘스트 방향성 검토용 샘플 생성 |

### 1.4 핵심 가치 (USP)
1. **속도**: 퀘스트 1개 생성에 30초 이내
2. **일관성**: 정해진 포맷으로 출력하여 즉시 실무 활용 가능
3. **커스터마이징**: 장르/테마/난이도별 맞춤 생성
4. **내보내기**: JSON/엑셀 형식으로 데이터 테이블 연동

## 2. 기능 정의

### 2.1 핵심 기능 (MVP)

| 기능 | 설명 | 우선순위 |
|------|------|---------|
| 장르 선택 | RPG, 액션, 시뮬레이션, 어드벤처 등 | P0 |
| 테마 입력 | 세계관/배경 설정 (예: "판타지 왕국", "사이버펑크 도시") | P0 |
| 난이도 설정 | 1~5단계, 보상 밸런스에 반영 | P0 |
| 퀘스트 타입 | 메인/서브/일일/반복 | P0 |
| 퀘스트 생성 | LLM 호출하여 퀘스트 생성 | P0 |
| 결과 출력 | 퀘스트명, 목표, 보상, NPC 대사 표시 | P0 |

### 2.2 부가 기능

| 기능 | 설명 | 우선순위 |
|------|------|---------|
| JSON 내보내기 | 생성된 퀘스트를 JSON 파일로 다운로드 | P1 |
| 엑셀 내보내기 | 실무 데이터 테이블 형식으로 내보내기 | P1 |
| 히스토리 | 생성한 퀘스트 목록 세션 내 유지 | P1 |
| 재생성 | 결과가 마음에 안 들면 다시 생성 | P1 |

### 2.3 향후 확장 기능 (v2.0)

| 기능 | 설명 | 우선순위 |
|------|------|---------|
| 퀘스트 체인 생성 | 연속 퀘스트 자동 설계 | P2 |
| 보상 밸런스 계산기 | 난이도 대비 보상 적정성 분석 | P2 |
| 다국어 출력 | 한/영/일 동시 생성 | P2 |
| 커스텀 프롬프트 | 고급 유저용 프롬프트 직접 편집 | P2 |

## 4. 데이터 구조

### 4.1 입력 데이터

```json
{
  "genre": "RPG",
  "theme": "중세 판타지 왕국",
  "difficulty": 3,
  "quest_type": "main",
  "language": "ko"
}
```

### 4.2 출력 데이터 (퀘스트 포맷)

```json
{
  "quest_id": "Q001",
  "quest_name": "잃어버린 왕관",
  "quest_type": "main",
  "difficulty": 3,
  "npc": {
    "name": "왕실 기사단장 로빈",
    "location": "왕성 대전"
  },
  "objective": {
    "type": "retrieve",
    "target": "황금 왕관",
    "location": "도적단 소굴",
    "kill_count": null
  },
  "rewards": {
    "gold": 500,
    "exp": 1200,
    "items": ["기사단 문장"]
  },
  "dialogue": {
    "accept": "용사여, 왕국의 보물인 황금 왕관이 도적들에게 탈취당했소...",
    "progress": "도적단 소굴은 북쪽 숲 끝에 있소. 조심하시오.",
    "complete": "역시 당신이었군요! 왕국을 대신하여 감사드리오."
  },
  "prerequisites": [],
  "next_quest": "Q002"
}
```

### 4.3 엑셀 출력 포맷

### 5.1 개발 환경

| 항목 | 선택 | 이유 |

## 7. 프롬프트 설계 (초안)

```
당신은 게임 퀘스트 디자이너입니다.

다음 조건에 맞는 퀘스트를 생성해주세요:
- 장르: {genre}
- 테마/세계관: {theme}
- 난이도: {difficulty}/5
- 퀘스트 타입: {quest_type}

출력 형식 (JSON):
{
  "quest_name": "퀘스트 이름",
  "npc_name": "NPC 이름",
  "npc_location": "NPC 위치",
  "objective": "목표 설명",
  "rewards": {
    "gold": 숫자,
    "exp": 숫자,
    "items": ["아이템명"]
  },
  "dialogue": {
    "accept": "수락 시 대사",
    "complete": "완료 시 대사"
  }
}

조건:
1. 난이도가 높을수록 보상을 증가시켜주세요
2. 테마에 맞는 세계관 용어를 사용해주세요
3. NPC 대사는 캐릭터성이 느껴지도록 작성해주세요
4. JSON만 출력하고 다른 설명은 하지 마세요
```
